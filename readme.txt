############################################
How to convert .ipynb to .py
############################################
1) CTRL + P ---> open search bar
2) Enter ">" and select Jupyter: Export to Python script


############################################
How to package .py file to .exe
############################################
1) Open shell or terminal
2) Enter "pyinstaller xxx.py" and wait for the process
	2.a) If you want .exe renamed: "pyinstaller --name yyy xxx.py"
	2.b) If you your file smaller, copy "upx.1" & "upx.exe"
	     to your progect root
3) Move .ui file to dist folder
